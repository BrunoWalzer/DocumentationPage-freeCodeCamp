HTML Introduction

HTML uses "Markup" to annotate text, images, and other content for display in a web browser. It is the most basic building block of the web. It defines the meaning and structure of web content.

History of HTML

HTML was created in 1991 by Tim Berners-Lee to structure web documents. Over time, it evolved through several versions, with HTML5 being the latest standard.

Top 3 HTML tags

The <"html"> tag defines the document structure, <"head"> contains metadata, and <"body"> holds the visible content.

  • <html>
  • <body>
  • <head>
<html> - Defines the structure of an HTML document and acts as the root element. <body> - Contains metadata such as the document title, character set, styles, and scripts. <head> - Holds the visible content of the web page, including text, images, and other elements.
Good pratices in HTML

Use semantic tags, maintain proper indentation, and ensure accessibility with attributes like alt for images.

CSS introduction

CSS (Cascading Style Sheets) is used to style HTML elements, separating design from structure for better maintainability.

History of CSS

CSS was introduced in 1996 to standardize web styling, evolving into versions like CSS3, which brought animations and flexbox.

Top 3 CSS commands

color defines text color, margin controls spacing around elements, and display determines how an element is rendered.

  • Color
  • Margin
  • Display
Color - Defines the text color of an element. Margin - Controls the space around an element, creating separation from other elements. Display - Determines how an element is rendered, such as block, inline, or flex.
Good pratices in CSS

Use external stylesheets, apply classes instead of inline styles, and follow a consistent naming convention.

JavaScript introduction

JavaScript is a programming language used to create interactive and dynamic web content, running in the browser.

History of JavaScript

Created in 1995 by Brendan Eich, JavaScript became essential for web development, evolving with ES6 and modern frameworks.

Top 3 JavaScript tips

Use "const" and "let" instead of "var", avoid global variables, and optimize loops for performance.

Const – Declares a variable that cannot be reassigned after its initial value is set. It ensures immutability for primitive values and is best used for constants that shouldn’t change. Let – Declares a variable with block scope, meaning it is only accessible within the {} where it was defined. Unlike const, it can be reassigned. Var – Declares a variable with function scope, meaning it is accessible throughout the entire function.
Good pratices in JS

Write clean, readable code, use comments wisely, and handle errors properly with try...catch.
